說明超連結的target 與title屬性，並舉一個html的例子

用target是指定打開連結文字的位置


用title是指定了有關元素的額外信息。 當滑鼠移動到元素上時，信息最常顯示為工具提示


簡單來說就是滑鼠移到要點的連結上時,還會跑出白色的框框,上面寫著顯示所打的字


範例The target 對象是 GOOGLE

說明href中以下幾個屬性


請問href 是什麼的縮寫 = href: Hypertext Reference


target = _blank 就是點下連結時會跑出一個新的分頁中來開啟新網頁


target = _self 就是點下連結時只會在自己本身的分頁來開新網頁


title 就是 滑鼠移到要點的連結元素上時，信息顯示為工具提示
